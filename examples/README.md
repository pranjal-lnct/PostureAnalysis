# Example Analysis Output

This folder contains sample output from the AI Posture Analysis application.

## Contents

### sample-analysis/
A complete posture analysis run including:

- **report.html** - Interactive HTML report with posture assessment and exercise recommendations
- **analysis.json** - Raw JSON data from the Gemini AI analysis
- **Images** - The four views used for analysis (front, left, right, back)

## Viewing the Report

To view the sample report, open `sample-analysis/report.html` in your web browser. The report includes:

- Overall posture score
- Detailed analysis of each body region
- Severity ratings for identified issues
- Personalized exercise recommendations
- Visual indicators and professional-grade assessment
